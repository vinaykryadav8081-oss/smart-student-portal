# Smart Student Portal

## Project Overview

Smart Student Portal is a web-based application developed using Django. It helps manage student information, attendance records, and academic marks through a simple and user-friendly interface.

## Features

- Student Registration and Login
- Student Dashboard
- Attendance Management
- Marks Management
- Admin Panel

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Conclusion

The Smart Student Portal simplifies student management by integrating attendance, marks, and authentication into a single web application.
